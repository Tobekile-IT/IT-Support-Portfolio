Service Desk Simulation 1: Customer Support PC Will Not Turn On 
OBJECTIVE: 
Investigate a high-priority Service Desk ticket where a customer support employee reported that their 
desktop computer would not power on. 
TICKET INFORMATION: 
•Priority: Critical 
•Department: Customer Support 
•Issue Reported: Desktop will not turn on 
•Business Impact: Customer support agent unable to take calls. 
INVESTIGATION  PERFORMED: 
1. Reviewed the ticket information. 
2. Read the user’s description. 
3. Connected to the user’s computer using Remote Desktop. 
4. Verified Windows was running normally. 
5. Checked that the computer was responsive. 
6. Opened the company Directory. 
7. Verified the user’s identity and department. 
8. Reviewed the company chat. 
9. Communicated findings with the user. 
FINDINGS: 
•Remote Desktop connected was successful. 
•Windows operating system was functioning normally. 
•No hardware or operating system errors were observed. 
•User identity matched the ticket information. 
•The computer was online and operational. 
•The user later confirmed that the monitor was working again. 
ROOT CAUSE: 
The issue was not a computer hardware failure. 
The investigation indicated the problem was most likely related to the user’s monitor or local display 
setup.  
RESOLUTION: 
Verified the computer was healthy using Remote Desktop. 
The user later confirmed that the monitor was functioning correctly and normal operation was restored. 
SKILLS PRACTICED: 
•Incident prioritization 
•Ticket investigation 
•Remote Desktop troubleshooting 
•User verification 
•Active Directory / Directory lookup 
•Critical thinking 
•User communication 
•Incident documentation 
•Root cause analysis 
WHAT I LEARND: 
This simulation taught me that user reports should always be verified before drawing conclusions. 
I also learned the important of checking all communication channels before closing a ticket. Even when 
the technical issue appears resolved, confirming the solution with the user is an important part of the 
Service Desk process.
